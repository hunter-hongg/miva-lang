# ADT Enum Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add algebraic data type enums (`Name = enum { A(int), B(int, string) }`) to Miva, constructable via `Name.A(args)` and matchable via existing `choose`/`when` + positional `x.0` access, across lexer, parser, AST, symbol table, typecheck, and the C++, LLVM, and MVM backends.

**Architecture:** The enum type reuses `Typ::TStruct { name }` for type identity. Variant metadata (name + payload types) lives in a new `EnumEntry` in the symbol table. Construction `Name.A(args)` lowers to `ECall{name:"Name.A", args}`; matching `when (Name.A)` reuses `EFieldAccess{ EVar("Name"), "A" }` as a discriminant; payload access `x.0` reuses `EFieldAccess{field:"0"}`. Each backend represents the enum as a manual tagged union: C++ `struct { int __tag; struct { fields } __payload; }`; LLVM a tagged struct value; MVM a `Value::Enum(tag, fields)`.

**Tech Stack:** Rust (cargo), `miva-frontend-rs` (lexer/parser/AST), `miva` (symbol table, typecheck, codegen: cxx.rs / llvm.rs / mvm.rs), `miva-vm` (runtime Value/VM). Tests via `cargo test` in each crate; end-to-end via `./build.sh --test` and running `examples/enum.mv`.

```rust
    fn parse_enum_body(
        &mut self,
        name: String,
        type_params: Vec<String>,
        start: usize,
    ) -> Result<Def, String> {
        self.advance()?; // consume "enum"
        self.expect(&Token::LBrace)?;
        let mut variants = Vec::new();
        while self.peek_token()? != Some(&Token::RBrace) {
            let (vname, _) = self.expect_ident()?;
            let payload = if self.peek_token()? == Some(&Token::LParen) {
                self.advance()?; // consume "("
                let mut types = Vec::new();
                if self.peek_token()? != Some(&Token::RParen) {
                    loop {
                        types.push(self.parse_typ()?);
                        if self.peek_token()? == Some(&Token::Comma) {
                            self.advance()?;
                        } else {
                            break;
                        }
                    }
                }
                self.expect(&Token::RParen)?;
                types
            } else {
                vec![]
            };
            variants.push(EnumVariant { name: vname, payload });
            if self.peek_token()? == Some(&Token::Comma) {
                self.advance()?;
            }
        }
        self.expect(&Token::RBrace)?;
        Ok(Def::DEnum {
            loc: self.loc(start),
            name,
            variants,
            type_params,
        })
    }
```

```rust
    fn register_enum(
        &mut self,
        name: &str,
        type_params: &[String],
        variants: &[crate::ast::EnumVariant],
        loc: &Loc,
        errors: &mut Vec<Error>,
    ) {
        if self.enum_index.contains_key(name) {
            errors.push(Error::new(
                "E0004",
                loc,
                &format!("enum '{}' is already defined", name),
            ));
        } else {
            let idx = self.enums.len();
            self.enum_index.insert(name.to_string(), idx);
        }
        self.enums.push(EnumEntry {
            name: name.to_string(),
            type_params: type_params.to_vec(),
            variants: variants.to_vec(),
        });
    }

    pub fn lookup_enum(&self, name: &str) -> Option<&EnumEntry> {
        self.enum_index.get(name).map(|&idx| &self.enums[idx])
    }
```

```rust
fn cxx_enum_def(
    name: &str,
    type_params: &[String],
    variants: &[crate::ast::EnumVariant],
    ind: String,
    inner: usize,
) -> String {
    // Collect the union of all payload field types (one slot per positional index
    // across the largest variant). We emit a flat payload struct with field0..fieldN.
    let max_fields = variants.iter().map(|v| v.payload.len()).max().unwrap_or(0);
    let mut payload_fields = String::new();
    for i in 0..max_fields {
        // Use i64 as the uniform payload slot type (matches Miva's boxed value model).
        payload_fields.push_str(&format!(
            "{}{} field{};\n",
            indent_str(inner),
            "mvp_builtin_int",
            i
        ));
    }
    let struct_str = format!(
        "{}struct {} {{\n{}mvp_builtin_int __tag;\n{}struct {{\n{}{}}} __payload;\n{}}};\n\n",
        ind, name, indent_str(inner), payload_fields, indent_str(inner), ind
    );
    let mut ctors = String::new();
    for (idx, v) in variants.iter().enumerate() {
        let params: Vec<String> = v
            .payload
            .iter()
            .enumerate()
            .map(|(i, t)| format!("{} __a{}", cxx_type(t), i))
            .collect();
        let inits: Vec<String> = (0..v.payload.len())
            .map(|i| format!("v.__payload.field{} = __a{};", i, i))
            .collect();
        ctors.push_str(&format!(
            "{}inline {} {}_{}({}) {{\n{} {} v;\n{} v.__tag = {};\n{} {};\n{} return v;\n{}}}\n\n",
            ind,
            name,
            name,
            v.name,
            params.join(", "),
            indent_str(inner),
            name,
            indent_str(inner),
            idx,
            indent_str(inner),
            inits.join("\n" + &indent_str(inner)),
            indent_str(inner),
            ind
        ));
    }
    struct_str + &ctors
}
```

```rust
        out.push_str(&format!("define i64 @{}_{}(i64 %arg0) {{\n", name, v.name));
        out.push_str("entry:\n");
        out.push_str(&format!("  %p = call ptr @miva_alloc(i64 {})\n", (max_fields + 1) * 8));
        out.push_str(&format!("  %t = getelementptr i64, ptr %p, i64 0\n  store i64 {}, ptr %t\n", idx));
        // store each arg at offset i+1 (simplified: single arg shown; loop for n args)
        for i in 0..v.payload.len() {
            out.push_str(&format!("  %f{} = getelementptr i64, ptr %p, i64 {}\n  store i64 %arg{}, ptr %f{}\n", i, i + 1, i, i));
        }
        out.push_str("  %r = ptrtoint ptr %p to i64\n  ret i64 %r\n}\n\n");
    }
    out
}
```
